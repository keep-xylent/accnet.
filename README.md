## 🚀 Fitur Utama

- 🔍 **Limited-Only Audit**: Fokus 100% pada item Limited untuk perhitungan Networth yang akurat.
- 📈 **Market Intelligence**: Integrasi data real-time dari Rolimons untuk indikasi **Demand** (Amazing, High, dll).
- 💱 **Multi-Currency Node**: Konversi nilai aset secara instan ke **Robux (R$)**, **USD ($)**, dan **IDR (Rp)**.
- 📊 **RAP Analysis History**: Grafik tren Networth 7 hari yang konsisten dan deterministik per user.
- 📱 **Fully Responsive**: Tampilan premium yang optimal baik di Desktop maupun perangkat Mobile (HP).
- ⚡ **High Performance**: Menggunakan batch API untuk sinkronisasi data dalam waktu kurang dari 2 detik.

## 🛠️ Teknologi

- **Backend**: Python, Flask
- **Frontend**: Vanilla JS, CSS, HTML5
- **Charts**: ApexCharts
- **APIs**: Roblox Web APIs, Rolimons Item API

## 🛡️ Keamanan & Privasi

- Aplikasi ini **tidak** memerlukan cookie atau password akun Roblox Anda.
- Hanya menggunakan API publik untuk mengambil data inventory publik.
